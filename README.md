## 📌 Project Overview  
This project focuses on building a **basic NLP-based chatbot** using Python.  
The chatbot processes user input, understands intent using Natural Language Processing techniques, and generates appropriate responses.

The goal of this assignment is to demonstrate foundational knowledge of:
- Natural Language Processing (NLP)
- Text preprocessing
- Intent recognition
- Rule-based conversational systems  

---

## ⚙️ Features  
- 🧠 Text preprocessing (tokenization, cleaning, normalization)  
- 💬 Rule-based chatbot responses  
- 🔍 Pattern matching for intent detection  
- 📊 Simple conversational flow  
- 🧪 Interactive testing via notebook  

---

## 🛠️ Technologies Used  
- Python 3.x  
- Jupyter Notebook  
- NLTK (Natural Language Toolkit)  
- NumPy / Pandas
